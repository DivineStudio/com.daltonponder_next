# Dalton Ponder | Full-Stack Developer & Cybersecurity Expert

Professional portfolio of Dalton Ponder - Full-stack developer and cybersecurity expert crafting secure, scalable solutions.

## Overview

This repository contains the source code for the personal portfolio website of Dalton Ponder. It showcases professional credentials, skills, testimonials, and contact information. The application is built with modern web technologies focusing on performance, accessibility, and security.

## Tech Stack

- **Framework**: [Next.js 16](https://nextjs.org/) (App Router)
- **UI Library**: [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **Animations**: [Motion](https://motion.dev/)
- **Internationalization**: [next-intl](https://next-intl-docs.vercel.app/)
- **Theming**: [next-themes](https://github.com/pacocoursey/next-themes)
- **Typography**: IBM Plex Mono, IBM Plex Sans, IBM Plex Serif

## Project Structure

The main application code is located in the `website` directory.
- `website/app`: App Router pages and layouts
- `website/components`: Reusable UI components and sections
- `website/public`: Static assets

## Getting Started

To run the development server locally:

1.  Navigate to the website directory:
    ```bash
    cd website
    ```

2.  Install dependencies:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  Start the development server:
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

To create a production build:

```bash
cd website
npm run build
npm start
```
